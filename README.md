# Logical replication with PostgreSQL

```
 -------------------          -----------------------------        
 | postgres-master |          | postgres-payments         |
 |-----------------|          |----------------------------
 | payments        |--------->| payments                  |
 |                 |------|   -----------------------------
 | clients         |---|  |   
 -------------------   |  |   -----------------------------
                       |  |   | postgres-payments2        |
                       |  |   |---------------------------|
                       |  |-->| payments                  |
                       |      -----------------------------
                       |
                       |      -----------------------------
                       |      | postgres-clients-payments |
                       |      |---------------------------|
                       |----->| clients                   |
                              | payments                  |
                              -----------------------------
```
